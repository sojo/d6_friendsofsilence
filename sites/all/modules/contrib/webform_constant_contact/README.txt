Webform Constant Contact Drupal Module

---------------------------------------------------
REQUIREMENTS
---------------------------------------------------

Before installing the module please ensure you have the constant contact module installed and working.

You need webform module version 6.x-3 or above to use this module.
You will also need COnstant contact module version 6.x-2.1 or above.

---------------------------------------------------
INSTALLATION
---------------------------------------------------

To install the module upload all files to your /sites/all/modules/ folder.

Login to your drupal admin panel and browse to the manage modules page (/admin/build/modules), check the checkbox next to the "Webform constant contact" module and the click the save configuration button, the module is under the "Webform" section.

The module is now enabled and you can use the webform component.
